# Master Thesis Code
## Comparison of Standard Fibers and Novel Optical Probes: 
## Illumination Profiles and Photothermal Effects in Liquid Brain Phantoms
Ellen Vanhulle & Gianni Brulez — Ghent University, 2025-2026

This repository contains all custom code used for data acquisition, 
processing, and statistical analysis in the master's thesis.

## Repository structure
- 1_laser_modulation: Python GUI for laser control via Arduino
- 2_FBG_interrogator: FBG sensor data acquisition scripts
- 3_light_distribution: MATLAB scripts for image processing and 
  contour analysis
- 4_temperature_analysis: MATLAB scripts for statistical analysis 
  and figure generation

## Requirements
- Python 3.x: tkinter, pyserial
- MATLAB
