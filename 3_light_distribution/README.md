MATLAB scripts for processing raw side- and bottom-view images acquired 
at 488 nm and 405 nm for all six optical fibers and probes.

Files:
- Absolute_intensity_and_intensity_contour_profiles.m: processes raw 
  images to generate absolute intensity profiles and iso-intensity 
  contour profiles
- ANOVA_5%_and_50%_intensity_contour_areas.m: statistical comparison 
  of 5% and 50% contour areas between fiber and probe types using
  ANOVA with Tukey post-hoc correction
- Graphs_and_barplots_intensity_contour_area.m: generates bar plots 
  comparing contour areas across fibers and probes
